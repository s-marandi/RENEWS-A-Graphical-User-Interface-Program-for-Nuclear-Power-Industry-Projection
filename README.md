# RENEWS

**Reactor Expansion & Nuclear Employment Workforce Simulator**  

---

## About

RENEWS is an **open-source graphical simulation tool** for the nuclear power industry.  
It links **reactor deployment scenarios** with **fuel cycle material demand** and **workforce requirements**, helping policymakers, researchers, and industry stakeholders explore “what-if” futures.  

The tool uses **Monte Carlo simulations** to quantify uncertainty and provides clear visual outputs to support long-term planning toward **2050 clean energy goals**.  

---

## Features

- Intuitive **graphical user interface (GUI)**  
- Flexible **scenario setup** (license extensions, growth functions, workforce factors)  
- **Probabilistic outputs** instead of fixed forecasts  
- Charts for capacity, workforce, and fuel cycle projections  
- Customizable datasets and assumptions
